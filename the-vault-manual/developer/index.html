Developer + Operator Manual (v4)

This manual is task-first and production-safe. It preserves required runtime anchors while improving practical execution flow.

Quick Start: install dependencies from requirements.txt, then run pytest manual gates before any publish action.

1) Runtime Ownership Map

  • vault_brain.py / Vault_Brain: orchestration and mission-state governance.
  • TaskMaster_Brain.py / TaskMaster_Brain: planning, decomposition, and queue execution.
  • Consensus_Engine/: decision arbitration and policy enforcement.
  • Contextual_Thinking/: context memory synthesis and retrieval.
  • Toolkit/omni_router.py: model and tool routing defaults.
